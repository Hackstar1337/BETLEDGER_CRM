Test: Same Bank Name with Different Account Holders

✅ Current Implementation

Account Holder Name: Must be unique (cannot duplicate)

Bank Name: Can be duplicated (multiple accounts can use same bank)

Account Number: Must be unique per account

Test Scenarios

Scenario 1: Same Bank, Different Holders (Should Work ✅)

Account 1:

  • Account Holder: John Doe
  • Bank Name: State Bank of India
  • Account Number: SBI001

Account 2:

  • Account Holder: Jane Smith
  • Bank Name: State Bank of India (same bank)
  • Account Number: SBI002

Expected Result: Both accounts should be created successfully

Scenario 2: Same Holder, Different Banks (Should Fail ❌)

Account 1:

  • Account Holder: John Doe
  • Bank Name: State Bank of India
  • Account Number: SBI003

Account 2:

  • Account Holder: John Doe (same holder)
  • Bank Name: ICICI Bank (different bank)
  • Account Number: ICICI001

Expected Result: Second account should be rejected with error about duplicate account holder name

⚠️ If You're Experiencing Issues

If you're unable to create accounts with the same bank name, it might be because:

  1. The database migration for the unique constraint hasn't been applied yet
  2. There might be additional validation in the frontend
  3. The account number might be duplicated (account numbers must be unique)
