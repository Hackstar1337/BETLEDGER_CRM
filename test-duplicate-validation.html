Test Duplicate Account Holder Name Validation

Test Instructions

1. Open the Casino Panel application in another tab

2. Navigate to Bank Accounts page

3. Click "Add Account" button

4. Try to create two accounts with the same Account Holder Name

5. The second attempt should show an error message

Test Data

First Account:

  • Account Holder Name: Test Duplicate User
  • Account Number: TEST001
  • Bank Name: Test Bank
  • Account Type: Deposit
  • Opening Balance: 1000

Second Account (should fail):

  • Account Holder Name: Test Duplicate User (same as above)
  • Account Number: TEST002
  • Bank Name: Another Bank
  • Account Type: Deposit
  • Opening Balance: 2000

Expected Results

✅ First account should be created successfully

✅ Second account should show error: "Account holder name 'Test Duplicate User' already exists. Please use a different name."
